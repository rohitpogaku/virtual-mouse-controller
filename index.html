AI Virtual Mouse and Keyboard

An AI Virtual Mouse and Keyboard System

Install v1.3.0

This project proposes a way of controlling the position of the mouse cursor with the help of a fingertip without using any electronic device. The proposed system will require only a webcam as an input device.

The computer's camera will scan the image of various movements made by a individual's hand, and the mouse or pointer will move in response to the movement of the gestures, including doing right and left clicks using distinct gestures. Similarly, different gestures can be used to control the keyboard, such as a one-finger gesture to choose an alphabet etc.

Objective

  1. To develop an alternative to the regular and traditional mouse system to perform and control the mouse functions
  2. Creating a virtual Human Computer Interaction(HCI) device such as mouse using a webcam and computer vision techniques can be an alternative way for the touch screen.

Modules Description

Hand Tracking Module

This is the building block of the project upon which the entire architecture & implementation is done.

To build this module Mediapipe By Google has been used in order to detect whether a palm is present in the given input frame & if it is present then using some methods provided in the mediapipe module calculation of the 21 landmarks of our hand is done along the with their x,y & z coordinates & are stored in an array.

Once this is done we go forward & plot the points of hand landmarks & lines connecting so the user can be assured that the hand has been detected.

Source: Google MediaPipe Hands

Mouse Module

The mouse module uses all the calculations & results from the Hand Tracking Module to detect the gestures & then according to the gestures detected it then performs the mouse functions like left click, right click etc.

Algorithm

Step 1:Take the input frame from the webcam.

Step 2:If Palm is detected then go to Step 3, otherwise to Step 1.

Step 3:Calculate the coordinates of the Landmarks of Hand Points & arrange it in an array.

Step 4:Use the coordinates of respective fingerâ€™s hand landmarks in order to detect the status of each finger.

Step 5:If Only Index & Middle Finger are in up position, then mouse goes to hover mode & then back to Step 1.

Step 6:If Index, Middle & Third Fingers are in up position, then mouse goes to Scrolling Mode & then back to Step 1.

Step 7: If Thumb is in up position, then mouse goes to Clicking mode.

Step 7a: If the index finger is down, then it performs a left click operation.

Step 7b: If Thumb is in up position, then mouse goes to Clicking mode.

Third Party Libraries Used

Python programing language is mostly used for developing this kind virtual mouse system, and OpenCV library is additionally used for contract computer vision.

The system makes use of the well-known python libraries like MediaPipe and Autopy for the tracking of the hands and fingertip in realtime and also, Pyput, and PyAutoGUI packages were used for tracking the finger traveling in the screen for performing operations like left-click, right-click, drag and scrolling functions.
